---------------------------------------------------------
AUTO-MARKETPLACE SETUP GUIDE
---------------------------------------------------------

1. START BACKEND (BE)
   - Go to: auto-marketplace-api
   - Run: npm run dev
   - Server runs on: http://localhost:5000

2. START NGROK (For Mobile Testing)
   - Run: npx ngrok http 5000
   - Copy the "Forwarding" URL (e.g., https://xxxx.ngrok-free.app)

3. UPDATE FRONTEND CONFIG
   - Open: auto-marketplace-app/src/services/api.js
   - Replace the URL in `baseURL` with your copied ngrok URL.

4. START FRONTEND (FE)
   - Go to: auto-marketplace-app
   - Run: npx expo start --tunnel
   - Scan the QR code to open in Expo Go.
